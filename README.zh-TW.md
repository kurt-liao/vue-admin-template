### 技術支援

- MVVM框架：vue v3
- 工程化管理：vite v2
- UI框架：element-plus
- 路由管理：vue-router v4
- 狀態管理：vuex v4
- 資源請求：axios
- 其他工具：@vueuse/core

### 使用

1. 下載專案

   ```
   git clone https://github.com/cmdparkour/vue-admin-box.git
   ```

   

2. 安装套件，可使用 npm 或 yarn

   ```
   npm install
   ```

   

3. 啟動

   ```
   npm run dev 或 npm run start
   ```

   

4. 打包

   ```
   npm run build
   ```
